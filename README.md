# Salvemos al Perú - Ate

Sitio estático base para la presencia distrital de `Salvemos al Perú - Ate`.

## Estructura

- `index.html`: home principal
- `propuestas.html`: propuestas distritales
- `regidores.html`: equipo y regidores
- `actividades.html`: trabajo territorial
- `documentos.html`: biblioteca institucional
- `fiscalizacion.html`: buzón ciudadano
- `styles.css`: estilos globales
- `script.js`: navegación responsive

## Publicación

El proyecto queda listo para publicarse como sitio estático en:

- GitHub Pages
- Vercel
- Netlify

Cuando el dominio `salvemosalperu-ate.org.pe` apunte al hosting elegido, solo hará falta configurar DNS y el dominio personalizado.
